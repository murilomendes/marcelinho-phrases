What is "E pra hoje marcelinho?"?

E pra hoje marcelinho? is about fun, at all. As I like to describe it, it is a fictional title generator for developers who doesn't know how to call themselves.

It plays with old technologies, hierarchy, and a lot of singular things in the development world - front-end, back-end, mobile, devops, databases...

After seeing crazy jobs and developers' titles on social media - specially on LinkedIn - I've decided to create this "hub", and merge them in a single website.

I was hugelly inspired by Design Titles, bullshit job titles generator, @kutyel/superb-developer-titles and the Silicon Valley Job Title Generator; obviously, by those unique titles as well.

This website is open source, and the code is on GitHub - take a look and don't be afraid to contribute!
